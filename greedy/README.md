- 잔돈 거슬러주기. 

  - 500,100,50,10 의 잔돈 중, 가장 적은 갯수의 잔돈을 주기 위해선, 가장 큰 500원으로 최대한 주는 것임으로, 가장 큰 동전부터 차례로 나눠간다. 

  ```python
  coins = [500,100,50,10]
  cnt = 0
  for coin in coins:
      cnt += change // coin #잔돈을 동전으로 나누고, 나머지를 버리고 몫만 반환 //
      change +% coin # 잔돈을 동전으로 나누고, 나머지를 반환
  ```
